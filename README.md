# AI Personal Expense Tracker

A simple personal expense tracking application built using Python and Streamlit.  
The application helps users record expenses, visualize spending patterns, and get basic budget insights.

---

## Overview

This project was built to simplify daily expense tracking.  
Users can add expenses, store them locally, and analyze their spending using charts and summaries.

The goal of this project is to demonstrate practical Python development, data handling, and visualization.

---

## Features

- Add expenses with date, description, amount, and category
- Rule-based expense categorization
- Category-wise spending visualization
- Daily spending trend analysis
- Basic budget insights
- Local data storage using SQLite

---

## Tech Stack

- Python
- Streamlit
- SQLite
- Pandas
- Matplotlib

## How to Run

1. Clone the repository:
   git clone https://github.com/Chandana-0225/AI-Personal-Expense-Tracker.git
   cd AI-Personal-Expense-Tracker
2. Install dependencies:
   pip install streamlit pandas matplotlib
3. Run the application:
   streamlit run app.py

   
---

## Logic Used

Expenses are categorized using simple keyword-based rules.  
Spending data is grouped and analyzed to generate charts and basic budget insights.
